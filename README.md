# 📊 Customer Shopping Analysis Dashboard (Power BI)

## 📌 Project Overview

This project is a Power BI dashboard built to analyze customer shopping behavior and generate actionable insights from sales data stored in MySQL.

The dashboard helps in understanding customer trends, purchase patterns, and overall business performance.

---

## 🎯 Objectives

* Analyze customer purchase behavior
* Identify high-value customers
* Track total revenue and average spending
* Evaluate customer ratings and satisfaction

---

## 🛠️ Tools & Technologies

* **Power BI** – Data visualization & dashboard creation
* **MySQL** – Data storage and querying
* **DAX (Data Analysis Expressions)** – Calculations and measures

---

## 📂 Dataset

The dataset includes:

* Customer ID
* Age / Age Group
* Purchase Amount
* Review Rating
* Subscription Status
* Previous Purchases

---

## 📈 Key Metrics (DAX Measures)

* **Total Revenue**
* **Average Purchase Amount**
* **Number of Customers**
* **Average Review Rating**

---

## 📊 Dashboard Features

* Revenue by Age Group
* Customer Segmentation
* Repeat Customer Analysis
* Ratings Overview
* Interactive Filters (Slicers)

---

## 🔍 Insights

* Identifies top contributing customer segments
* Helps understand spending patterns
* Tracks customer satisfaction via ratings

---

## 🚀 How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Refresh data (if MySQL connected)
4. Explore dashboard visuals
